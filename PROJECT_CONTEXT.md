# HasPaket — Proje Bağlamı

## Ne bu proje?

**HasPaket**, kurye ve teslimat süreçlerini yönetmek için geliştirilen bir web uygulamasıdır. Operasyonel paneller (admin, kurye) ve müşteri/operasyon giriş akışları hedeflenir.

## Marka ve ürün

| Öğe | Değer |
|-----|--------|
| Marka adı | HasPaket |
| Tür | Kurye / teslimat yönetim sistemi |
| Görsel dil | Premium, modern, startup tarzı |
| Ana renk | `#FF6B00` |
| Tema | Koyu arka plan, yumuşak glow, modern kart yapısı |
| Logo | Projede seçilmiş HasPaket logosu kullanılacak |

## Geliştirme sırası (yüksek seviye)

1. **Anasayfa** — İlk odak: marka, değer önerisi, CTA.
2. **Login** — Kimlik doğrulama ekranı.
3. **Admin paneli** — Yönetim arayüzü.
4. **Kurye paneli** — Sahadaki kurye arayüzü.

## Bu dosyanın amacı

Kod veya özellik eklenmeden önce burayı okuyarak proje hedefi, sıra ve tasarım çerçevesinin unutulmaması hedeflenir. Ayrıntılı kararlar `DESIGN_DECISIONS.md` içindedir.
